CASINO PRO - VERSION CLIENTE CON LICENCIA

SUBIR A GITHUB:
1) Subí TODOS estos archivos al repositorio:
   - index.html
   - manifest.webmanifest
   - sw.js
   - licencias.json
   - icon-192.png
   - icon-512.png

COMO ACTIVAR:
1) Abrí tu Panel Maestro de Licencias.
2) Elegí: CAS — Casino.
3) Poné cliente y vencimiento.
4) Tocá Crear licencia.
5) Copiá el bloque JSON.
6) Pegalo dentro de licencias.json.

IMPORTANTE:
- licencias.json debe quedar como una lista con corchetes [ ... ].
- Si agregás más clientes, separá cada bloque con coma.
- Si cambiás estado a "bloqueado", la app no entra.
- Si la fecha vence, la app queda bloqueada.
